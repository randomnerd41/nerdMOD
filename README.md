# nerdMOD
a FOSS python module for nerds!
made by randomnerd41

# how to use
put you "main.py" file in
the same folder as "nerdMOD.py"
at the very top of your python script
put: from nerdMOD import *

#funcs

1. clear() - clears the screen on linux or windows and even macOS (might work on freeBSD)

2. wait(wait time in seconds goes here!) - waits for as long as you put in seconds!

3. update_debian - updates any debian based linux system such as linux mint or ubuntu (needs sudo but will ask first) (only works on debian linux)

4. FOSS - prints "this program is FOSS" FOSS = free and opensource software
